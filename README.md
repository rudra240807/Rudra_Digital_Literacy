## Project Overview
This repository contains my Digital Literacy Portfolio created for the Digital Literacy course.  
The goal of this project is to show my understanding of responsible digital behavior, safe internet practices, and professional communication in online environments.

The portfolio includes five tasks covering topics like digital awareness, professional online profiles, coding platforms, communication etiquette, and cybercrime prevention.

## Task 1, Digital Literacy Infographic
I designed a visual infographic using Canva to explain the concept of digital literacy.  
The infographic highlights key topics such as safe internet usage, useful digital tools for students, and how to maintain a professional online presence.
